Структура проекта:
 * папка commands содержит файлы с текстом для команд чат бота;
 * папка reports содержит логи, историю запросов пользователей, а также пароль для админа;
 * папка standards содержит нормативные документы по оформлению пояснительной записки и прохождению нормоконтроля;
 * файл Config.py включает в себя датасет;
 * файл Final_bot.py включает в себя код чат бота в Telegram;
 * файл logging.conf включает в себя настройки логирования;
 * файл Models.py включает в себя код модели машинного обучения, алгоритма, основанного на расстоянии Левенштейна, "заглушек";
 * файл ML_model_test.ipynb включает в себя код с тестированием различных алгоритмов векторизации, классификации и выбором лучшей пары векторизатора и классификатора.
